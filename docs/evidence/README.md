# Evidence

Evidence records keep the skill names used when each experiment ran. Use this map when tracing historical results to current skills:

| Historical name | Current name |
|-----------------|--------------|
| `skill-creator` | `create-skill` |
| `spec-plan`, `capture-spec` | `create-spec` |
| `slice-spec` | `create-tasks` |
| `implement-slice` | `implement-task` |
| `ship-feature` | `deliver-feature` |
| `tldr`, `brevity` | `be-concise` |

Historical results support only the tested revision and invocation route. A force-loaded result does not establish autonomous triggering.
